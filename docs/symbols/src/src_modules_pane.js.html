319 				 *
320 				 * - To get more of an idea on how the panes objec structure should look
321 				 *   like see the validatePaneStructure function.
322 				 *
323 				 *  The example above's object structure looks like this:
324 				 *
325 				 * View(objectInstance).paneTree = Pane(ObjectInstance){
326 				 *   type: 'vertical', // There
327 				 *
328 				 *   width: '50%',     // or an integer for example if the screen was 800
329 				 *                     // pixels wide this could be 375 which means the
330 				 *                     // left pane will be 375 pixels wide and then the
331 				 *                     // right pane will take up the additional space
332 				 *                     // e.g 425 pixels. So the width specified for a
333 				 *                     // set of two panes is technically the width of the
334 				 *                     // first pane, and the second pane will take up the
335 				 *                     // rest of the space available.
336 				 *
337 				 *
338 				 *   // Notice that all the 'panes:' properties below can only have
339 				 *   // an array of two Panes. No less, no more. When you only need to 
340 				 *   // render one pane (for example just one big pane to the screen)
341 				 *   // the Pane object will not have any panes: but use tabs: instead.
342 				 *
343 				 *   panes: [
344 				 *
345 				 *			Pane(ObjectInstance){
346 				 *
347 				 *				type: 'horizontal',
348 				 *				height: '50%',
349 				 *				panes: [
350 				 *
351 				 *					Pane(ObjectInstance){
352 				 *						tabs: [Tab(ObjectInstance)] // no more panes, render whats in the tab.
353 				 *					},
354 				 *
355 				 *					Pane(ObjectInstance){
356 				 *						tabs: [Tab(ObjectInstance)] // no more panes, render whats in the tab.
357 				 *					},
358 				 *
359 				 *				]
360 				 *
361 				 *			},
362 				 *
363 				 *			Pane(ObjectInstance){
364 				 *				tabs: [Tab(ObjectInstance)]
365 				 *			}
366 				 *
367 				 *   ]
368 				 *
369 				 * }
370 				 *
371 				 * The desired result will be:
372 				 * [
373 				 *	Pane(ObjectInstance){
374 				 *		area: {x,y,width,height}
375 				 *	},
376 				 *	Pane(ObjectInstance){
377 				 *		area: {x,y,width,height}
378 				 *	},
379 				 *	Pane(ObjectInstance){
380 				 *		area: {x,y,width,height}
381 				 *	},
382 				 *	Pane(ObjectInstance){
383 				 *		area: {x,y,width,height}
