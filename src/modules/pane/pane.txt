View/Window/Panes/Tabs structure explination:

A view is a window, it can contain zero or more panes, when a view
has no panes, this is considered a no documents open state.

When a view contains one or more panes, each pane must contain either
one or more tabs, or one or more panes.

Panes must have either a pane or tab property inside of them.

Tabs are the last item in this structure and is where the component
will be rendered.
